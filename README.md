# 🛒 E-Commerce Website

## 📌 Description
A full-stack e-commerce web application built using HTML, CSS, JavaScript, PHP, and MySQL.  
Users can browse products, add items to cart, update quantities, and manage their shopping experience.

---

## 🚀 Features
- User Registration & Login
- Add to Cart functionality
- Update / Remove items from cart
- Dynamic total price calculation
- Admin panel to manage products
- Responsive UI design

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL
- Version Control: Git & GitHub

---

## 📂 Project Structure
- `/index.php` → Home page
- `/pages/cart.php` → Cart functionality
- `/admin/` → Admin panel
- `/css/` → Styling
- `/includes/` → Database connection
